LINK ALERTS — SETUP
===================

WHAT THIS DOES
--------------
When someone posts a link in #quick_fandoms or #quick_originals, the bot pings
the poster in #rule_infractions asking them to remove the link from their ad.


BEFORE YOU START
----------------
1. Turn on Discord Developer Mode:
   User Settings → Developer → Developer Mode (ON).
   This lets you right-click a channel and "Copy Channel ID".

2. Copy this channel ID (right-click channel → Copy Channel ID):
   • #rule_infractions 


CREATE THE "Link Alert" COMMAND
-------------------------------
1. YAGPDB dashboard → Custom Commands → New Command.
2. Trigger type: Regex
3. Trigger: (?i)https?://\S+
4. Channel Restrictions: Check off 'Only run in the following channels'
   then add #quick_fandoms and #quick_originals to the channel list below.
5. Open link_alert.go and copy its contents into the command's code box.
6. In that pasted code, set:
       {{ $infractionsChannel := 0 }}
   By replacing the 0 with a paste of #rule_infractions channel ID.
7. Save the command.


NOTES
-----

• Messages posted via the GIF / sticker picker usually contain an https link
  (e.g. a tenor.com URL) and will trigger an alert too.
