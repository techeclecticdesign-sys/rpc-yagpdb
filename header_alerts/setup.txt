HEADER ALERTS — SETUP
=====================

WHAT THIS DOES
--------------
When someone posts a Discord header in one of the one-on-one advert channels,
the bot pings the poster in #rule_infractions asking them to remove it.


BEFORE YOU START
----------------
1. Turn on Discord Developer Mode:
   User Settings → Developer → Developer Mode (ON).
   This lets you right-click a channel and "Copy Channel ID".

2. Copy this channel ID (right-click channel → Copy Channel ID):
   • #rule_infractions  (where the reminder is posted)


CREATE THE "Header Alert" COMMAND
---------------------------------
1. YAGPDB dashboard → Custom Commands → New Command.
2. Trigger type: Regex
3. Trigger: (?m)^#{1,3}\x20
4. Channel Restrictions: Check off 'Only run in the following channels'
   then add your one-on-one advert channels to the channel list below.
5. Open header_alert.go and copy its contents into the command's code box.
6. In that pasted code, set:
       {{ $infractionsChannel := 0 }}
   By replacing the 0 with a paste of #rule_infractions channel ID.
7. Save the command.
